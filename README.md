# ⚡ EV Customer Segmentation Dashboard

## 📌 Project Overview
The **EV Customer Segmentation Dashboard** is an interactive Power BI project designed to analyze customer demographics, electric vehicle (EV) adoption, purchasing behavior, and customer priorities. The dashboard helps identify high-value customer segments and provides actionable insights for targeted marketing and business decision-making.

## 🎯 Objectives
- Analyze customer distribution across different segments.
- Measure EV adoption among customers.
- Identify high-priority customers for marketing campaigns.
- Compare customer types, vehicle preferences, and income groups.
- Provide interactive filtering for detailed business analysis.

## 📊 Dashboard Features
- **KPI Cards**
  - Total Customers
  - EV Users
  - EV Adoption Percentage
  - High Priority Customers
  - Average Monthly Income

- **Interactive Visualizations**
  - Customer Type Distribution
  - EV Status Analysis
  - Vehicle Type Distribution
  - Customer Category Treemap
  - Priority Distribution
  - City-wise Customer Analysis
  - Customer Details Table

- **Interactive Filters (Slicers)**
  - City
  - Customer Type
  - Vehicle Type
  - Income Group
  - Priority

## 🛠 Tools & Technologies
- **Power BI Desktop**
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling

## 📈 Key Insights
- Monitors overall EV adoption across customers.
- Identifies customers with high purchasing potential.
- Compares customer segments based on income, vehicle type, and location.
- Enables business users to make data-driven marketing decisions.
- Provides an interactive dashboard experience through dynamic filtering.
